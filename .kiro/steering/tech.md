# 技術スタック

## アーキテクチャ

ローカルファースト・シングルページアプリケーション。バックエンド不要で、デバイス上でデータと画像を管理する。

## コア技術

- **言語**: TypeScript（strict モード、`any` 禁止）
- **フレームワーク**: 未定（React / Next.js 等、spec-design フェーズで決定）
- **ランタイム**: Node.js 20+

## 主要ライブラリ

spec-design フェーズで決定予定。以下の領域で選定が必要：

- アニメーション（ルーレット / スロット演出）
- カメラ / 写真撮影
- ローカルデータ永続化（IndexedDB 等）
- ルーティング

## 開発標準

### 型安全性

- TypeScript strict モード必須
- `any` 型の使用禁止、discriminated union を活用
- インターフェースで型定義を明示

### コード品質

- ESLint + Prettier による自動整形
- 具体的なルールセットは初期セットアップ時に決定

### テスト

- テストフレームワーク未定（Jest / Vitest 等）
- AAA パターン（Arrange-Act-Assert）を基本とする
- テストファイルは `*.test.*` または `*.spec.*` の命名規則

## 開発環境

### 必要ツール

- Node.js 20+
- npm / pnpm（パッケージマネージャーは初期セットアップ時に決定）

### 共通コマンド

```bash
# Dev:   未定
# Build: 未定
# Test:  未定
# Lint:  未定
```

## 重要な技術判断

- **ローカルファースト**: ログイン・クラウド同期なし。デバイス上で完結
- **カメラアクセス**: 表紙撮影のため必須
- **子ども向けUI**: 大きなボタン、表紙画像ドリブン、最小限のテキスト
